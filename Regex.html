Data Structure Documentation

struct regexpr_match_t

Type used to represent a match.

A match is represented as starting and ending offsets (soff and eoff) of the string to search in. This effectively gives you a substring which contains the match. Invalid matches are indicated when both soff and eoff are -1.

Data Fields
int eoff

The offset to the ending of the string where a match was found.

int soff

The offset to the starting of the string where a match was found.
